Tipos de datos en MySQL

varchar

se usa almacenar cadenas de caracteres. Una cadena es una secuenacia de caracteres. Se coloca entre comillas ( simples):'Hola'
El tipo ¨varchar¨ define una cadena de longitud variable en la cual determinamos el máximo de caracteres. Puede guardar hasta 255 caracteres. Para almacenar cadenas de haasta 30 caracteres, definimos un campo de tipo varchar(30).

int

Se usa para guardar valores numéricos enteros de -2000000000 a 2000000000 aproximadamente
Definimos campos de este tipo cuando quermos representar por ejemplo cantidades.
